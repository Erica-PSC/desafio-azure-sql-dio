# Anotações do Laboratório

## Azure

Microsoft Azure é uma plataforma de computação em nuvem que oferece serviços de armazenamento, bancos de dados, redes, segurança e hospedagem de aplicações.

## Resource Group

Um Resource Group é um contêiner lógico utilizado para organizar recursos relacionados dentro de uma assinatura Azure.

## SQL Server

O SQL Server no Azure funciona como um servidor lógico responsável por hospedar bancos de dados SQL.

## Azure SQL Database

Serviço de banco de dados relacional totalmente gerenciado pela Microsoft.

## Benefícios

* Alta disponibilidade
* Escalabilidade
* Segurança integrada
* Gerenciamento simplificado
* Integração com outros serviços Azure
