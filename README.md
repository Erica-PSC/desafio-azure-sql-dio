# Desafio DIO - Configuração de Banco de Dados no Microsoft Azure

## Descrição

Este repositório foi criado para documentar a realização do laboratório de configuração de uma instância de Banco de Dados SQL na plataforma Microsoft Azure.

O objetivo da atividade foi compreender o processo de criação, configuração e gerenciamento de recursos em nuvem utilizando os serviços disponibilizados pela Microsoft Azure.

## Recursos Criados

Durante a execução do laboratório foram criados os seguintes recursos:

### Grupo de Recursos

* Nome: rg-dio-lab

### Servidor SQL

* Nome: servidor-dio-erica
* Região: Central US

### Banco de Dados SQL

* Nome: banco-dio

## Passo a Passo Realizado

### 1. Acesso ao Portal Azure

Foi realizado o acesso ao Portal Microsoft Azure utilizando uma conta Microsoft com assinatura ativa.

### 2. Criação do Grupo de Recursos

Foi criado o grupo de recursos chamado "rg-dio-lab" para organizar os componentes do projeto.

### 3. Criação do Servidor SQL

Foi criado um servidor SQL contendo:

* Nome do servidor
* Usuário administrador
* Senha de autenticação
* Região de implantação

### 4. Criação do Banco de Dados

Foi criado o banco de dados "banco-dio" associado ao servidor SQL criado anteriormente.

### 5. Implantação dos Recursos

Após a validação das configurações, os recursos foram implantados com sucesso na plataforma Azure.

## Aprendizados

Durante a realização do laboratório foi possível compreender:

* Conceitos fundamentais de computação em nuvem.
* Organização de recursos através de Resource Groups.
* Criação de servidores SQL gerenciados.
* Implantação de bancos de dados em nuvem.
* Gerenciamento de recursos no Microsoft Azure.

## Evidências

As capturas de tela utilizadas para documentar o processo encontram-se na pasta images.

## Conclusão

A atividade permitiu adquirir experiência prática na utilização do Microsoft Azure para provisionamento de bancos de dados SQL, reforçando conceitos importantes relacionados à computação em nuvem e administração de serviços gerenciados.
